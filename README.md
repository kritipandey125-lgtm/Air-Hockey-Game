# 🏒 Air Hockey Game

A simple and fun Air Hockey game built using HTML, CSS, and JavaScript. Control your paddle, hit the puck, and score against your opponent!

## 🚀 Features
- Smooth gameplay
- Score tracking
- Responsive controls
- Restart game option

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎮 How to Play
- Move your paddle to hit the puck.
- Score goals against your opponent.
- The player with the highest score wins.
